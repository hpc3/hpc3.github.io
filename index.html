>the-stand

A full stack web app that serves as a public site for customers and management tool for employee's. React handles rendering out UI. Redux handles state management. A REST API built with Express.js handles communication between the frontend and mongoDB database.

>Tools

>Frontend

  • - React
  • - Redux
  • - styled-components
  • - Chart.js

>Backend

  • - Node.js
  • - Express.js
  • - MongoDB

>Features

>Customers

  • - View details about the produce stand
  • - View products and a live inventory count
  • - Leaves comments for produce stand employees

>Employees

  • - Login/Authentications
  • - Edit Produce Items [adjust stock quantity, name, price etc.]
  • - View and response to comments left by customers
  • - Submits and track sales/expenses
  • - View and alter charts related to submitted financial information

>HOPWA Rent Calculator

A full stack web app built for RDE Systems in conjunction with New Jersey Institute of Technology Capstone Program. It serves as a tool to assist caseworks in calculating benefits received by clients affiliated with the Housing Opportunities for Persons with AIDS (HOPWA) program.

>Tools

>Frontend

  • - React

>Backend

  • - MSSQL
  • - Coldfusion

>Features

>Caseworkers

  • - Add/Search clients
  • - View client information
  • - Collect data to create worksheets based on client data [i.e. income, expenses]
  • - Calculate benefits based on collected data
  • - View past worksheets
